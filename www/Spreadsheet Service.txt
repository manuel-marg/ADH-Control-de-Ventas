Spreadsheet Service

Organiza tus páginas con colecciones Guarda y categoriza el contenido según tus preferencias.


--------------------------------------------------------------------------------------------------------------------

Este servicio permite que las secuencias de comandos creen, accedan y modifiquen archivos de Hojas de cálculo de Google. Consulta también la [guía para almacenar datos en hojas de cálculo](https://developers.google.com/apps-script/storing_data_spreadsheets?hl=es-419).

A veces, las operaciones de hoja de cálculo se agrupan para mejorar el rendimiento, por ejemplo, cuando se realizan varias llamadas a un método. Si quieres asegurarte de que todos los cambios pendientes se realicen de inmediato, por ejemplo, para mostrar información a los usuarios mientras se ejecuta una secuencia de comandos, llama a [`SpreadsheetApp.flush()`](https://developers.google.com/apps-script/reference/spreadsheet/spreadsheet-app?hl=es-419#flush).

  * Tipo de datos que se muestra: void
  * Descripción breve: Muestra una ventana emergente en la esquina inferior derecha de la hoja de cálculo con el título y el mensaje determinados, que permanecen visibles durante un período determinado.
* Método: unhideColumn(column)
  * Tipo de datos que se muestra: void
  * Descripción breve: Muestra la columna en el rango determinado.
* Método: unhideRow(row)
  * Tipo de datos que se muestra: void
  * Descripción breve: Muestra la fila en el rango determinado.
* Método: updateMenu(name, subMenus)

  * Descripción breve: Actualiza un menú que agregó addMenu(name, subMenus).

### Métodos



* Método: findAll()
  * Tipo de datos que se muestra: Range[]
  * Descripción breve: Devuelve todas las celdas que coincidan con los criterios de búsqueda.
* Método: findNext()
  * Tipo de datos que se muestra: Range
  * Descripción breve: Muestra la siguiente celda que coincide con los criterios de búsqueda.
* Método: findPrevious()
  * Tipo de datos que se muestra: Range
  * Descripción breve: Muestra la celda anterior que coincide con los criterios de búsqueda.
* Método: getCurrentMatch()
  * Tipo de datos que se muestra: Range
  * Descripción breve: Muestra la celda actual que coincide con los criterios de búsqueda.
* Método: ignoreDiacritics(ignoreDiacritics)
  * Tipo de datos que se muestra: TextFinder
  * Descripción breve: Si es true, configura la búsqueda para que ignore los signos diacríticos durante la coincidencia; de lo contrario, la búsqueda coincide con los signos diacríticos.
* Método: matchCase(matchCase)
  * Tipo de datos que se muestra: TextFinder
  * Descripción breve: Si es true, configura la búsqueda para que coincida exactamente con el caso del texto de búsqueda. De lo contrario, la búsqueda usará la concordancia que no distingue mayúsculas de minúsculas de forma predeterminada.
* Método: matchEntireCell(matchEntireCell)
  * Tipo de datos que se muestra: TextFinder
  * Descripción breve: Si es true, configura la búsqueda para que coincida con todo el contenido de una celda. De lo contrario, la búsqueda se establece de forma predeterminada en la coincidencia parcial.
* Método: matchFormulaText(matchFormulaText)
  * Tipo de datos que se muestra: TextFinder
  * Descripción breve: Si es true, configura la búsqueda para que muestre coincidencias que aparezcan dentro del texto de la fórmula. De lo contrario, las celdas con fórmulas se consideran según su valor mostrado.
* Método: replaceAllWith(replaceText)
  * Tipo de datos que se muestra: Integer
  * Descripción breve: Reemplaza todas las coincidencias por el texto especificado.
* Método: replaceWith(replaceText)
  * Tipo de datos que se muestra: Integer
  * Descripción breve: Reemplaza el texto de búsqueda en la celda que coincide actualmente con el texto especificado y muestra la cantidad de ocurrencias reemplazadas.
* Método: startFrom(startRange)
  * Tipo de datos que se muestra: TextFinder
  * Descripción breve: Configura la búsqueda para que comience inmediatamente después del rango de celdas especificado.
* Método: useRegularExpression(useRegEx)
  * Tipo de datos que se muestra: TextFinder
  * Descripción breve: Si es true, configura la búsqueda para que interprete la cadena de búsqueda como una expresión regular. De lo contrario, la búsqueda interpreta la cadena de búsqueda como texto normal.
